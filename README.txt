he exercise in practicing modular design with closure from the 12/11 study group

The objective was to take some code you already wrote, or write new code, and spread out as much of it in typescript as you can to other files. Then add in import and exports to your main file. Then try to use closure to make it so you don't have any global variables that you don't need to.

For my version I decided to write up a to do app for node.js
